Voorstellen door aanpassen bestaande webpagina

Om kennis te maken met het vak Smart Connection hebben we de opdracht gekregen om van een bestaande webpagina de HTML code aan te passen en hiermee een eigen versie te maken van een bestaande website. Ik heb hier gekozen voor transfermarkt.nl. Dit is een site waar de waarde van voetballers weergegeven wordt op de markt, dit vond ik wel een ludieke manier om mezelf voor te stellen. Mijn grootste hobby is voetballen en vond ik het dus wel toepasselijk om mezelf op deze manier aan te bieden op de arbeidsmarkt. Onderstaand de bestaande website transfermarkt en een link naar mijn aangepaste versie.
