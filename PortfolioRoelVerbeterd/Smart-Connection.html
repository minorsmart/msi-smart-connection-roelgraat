Smart Connection

Bij het vak Smart Connection was het einddoel om deze portfolio te kunnen maken als een website met daarachter HTML en CSS code. Maar wat is nou precies HTML en CSS?
De combinatie van HTML en CSS samen vormt de front-end van een website, dat is de website hoe je hem nu ziet. In deze code achter zo'n website, vormt HTML eigenlijk het geschreven deel, oftewel de structuur, van de website en zorgt CSS voor de opmaak van de website.

Voorstellen door aanpassen bestaande webpagina

Om kennis te maken met het vak Smart Connection hebben we de opdracht gekregen om van een bestaande webpagina de HTML code aan te passen en hiermee een eigen versie te maken van een bestaande website. Ik heb hier gekozen voor transfermarkt.nl. Dit is een site waar de waarde van voetballers weergegeven wordt op de markt, dit vond ik wel een ludieke manier om mezelf voor te stellen. Mijn grootste hobby is voetballen en vond ik het dus wel toepasselijk om mezelf op deze manier aan te bieden op de arbeidsmarkt. Smart Connection <--> Smart Business

Smart Connection is eigenlijk heel breed en kan vanuit allerlei kanten benaderd worden, zo is het in deze minor vooral benaderd als het maken van een website met HTML en CSS code. Echter is er nog veel meer achter zo'n website wanneer deze in de het bedrijfsleven draait. Zo zit er hoogstwaarschijnlijk achter de meeste websites een database die gegevens van bijvoorbeeld klanten bijhoudt en deze ergens opslaat. Om me te verdiepen in de Smart Connection in combinatie met de Smart Business heb ik gekozen om de casus van AXA uit te werken, alleen dan te bekijken hoe dit op het gebied van connection allemaal opgezet kan worden. Dit is in onderstaand document te zien:

Casus axa - connection

Ook kunnen we is de status van Van den Broek bekijken op het gebied van Connection. Eigenlijk is Van den Broek zelf eigenlijk een firma die beveiligingssystemen levert aan andere bedrijven, hierdoor wordt er heel gebruik gemaakt van data op de cloud. Dit heeft als voordeel dat deze data altijd beschikbaar is voor Van den Broek ongeacht waar deze zijn, dit is zeer wenselijk en eigenlijk ook gewoon noodzakelijk. Bijvoorbeeld in onze drone toepassing maken wij ook gebruik van een applicatie op een webserver, dit zodat deze vanaf elke PC op het bedrijf te benaderen is zonder dat daarvoor eerst heel veel ge√Ønstalleerd hoeft te worden. Al deze dingen zijn vormen van Connection. De toekomst gaat steeds meer toe naar onderhoud op afstand en daarvoor is industrie 4.0 heel erg belangrijk door zoveel gebruik te maken van internet. Het is voor Van den Broek heel erg belangrijk om hierin mee te gaan omdat dit veel onnodige reiskosten kan besparen door onderhoud op afstand te kunnen regelen.
